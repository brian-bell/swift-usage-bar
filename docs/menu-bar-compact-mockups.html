Implementation notes

  • Two lines are visually possible only if the status item is rendered as a custom image or a tightly controlled custom label view; normal text sizing is too tall.
  • The two-line provider stack is the best first prototype: it preserves provider identity, stale markers, and the current pair grammar while cutting horizontal width substantially.
  • The matrix variants are more compact, but they require users to learn that each row is provider and each column is 5h/week.
  • The adaptive minimum variant is the most compact, but it changes the information model because weekly values disappear from the menu bar until the user opens the dropdown.
